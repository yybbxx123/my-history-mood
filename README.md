# 古代人格测评应用

一个使用原生 HTML、CSS 和 JavaScript 构建的现代待办事项应用。

## 功能特性

- ✅ 8 道人格测评题目
- ✅ 自动匹配古代人物原型
- ✅ 历史人物故事浏览
- ✅ 结果保存到本地
- ✅ 移动端风格界面

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript ES6+** - 交互逻辑
- **LocalStorage** - 本地保存测评结果

## 如何运行

1. 直接在浏览器中打开 `index2.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000

   # 使用 Node.js
   npx serve .
   ```

## 项目结构

```
├── index2.html   # 主页面入口
├── style.css     # 样式文件
├── script.js     # 逻辑代码
├── data.js       # 测评题库与人物数据
└── README.md     # 项目说明
```
