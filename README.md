# 简单待办事项应用

一个使用原生 HTML、CSS 和 JavaScript 构建的现代待办事项应用。

## 功能特性

- ✅ 添加新任务
- ✅ 标记任务完成
- ✅ 删除任务
- ✅ 本地存储（刷新页面数据不丢失）
- ✅ 响应式设计
- ✅ 现代 UI 设计

## 技术栈

- **HTML5** - 语义化结构
- **CSS3** - 现代样式和动画
- **JavaScript ES6+** - 类、箭头函数、模板字符串
- **LocalStorage** - 数据持久化

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve .
   ```

## 项目结构

```
├── index.html    # 主页面
├── style.css     # 样式文件
├── script.js     # 逻辑代码
└── README.md     # 项目说明
```

## 代码亮点

- 使用 ES6 类组织代码
- 事件委托和现代 DOM 操作
- CSS Grid/Flexbox 布局
- 渐变背景和过渡动画
- 数据持久化存储
